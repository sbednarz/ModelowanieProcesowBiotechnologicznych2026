### Dlaczego Python?

- **Darmowy i otwartoźródłowy** (open-source), wysokopoziomowy, ogólnego przeznaczenia język programowania skryptowego.  
- **Dojrzały (ponad 30 lat rozwoju)**, dobrze udokumentowany, z ogromną społecznością użytkowników i bardzo dużą popularnością.  
- **Główne zastosowania:** edukacja, obliczenia naukowe, przetwarzanie i wizualizacja danych, uczenie maszynowe, systemy wbudowane oraz wiele innych.
- Python jest jednym z najważniejszych języków w obszarze **AI, Machine Learning i Deep Learning**  (ze względu na biblioteki takie jak NumPy, Pandas, Scikit-learn, TensorFlow czy PyTorch).
